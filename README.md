#  Travel Website Frontend

A responsive travel website frontend built using HTML, CSS, and JavaScript.
This project showcases travel destinations, packages, and booking UI with a modern design.

# Features

• Beautiful homepage UI

• Responsive design (Mobile & Desktop)

• Travel destinations section

• Tour packages display

• Contact form

• Smooth navigation

• Clean & modern layout


# Technologies Used

HTML5

CSS3

JavaScript
